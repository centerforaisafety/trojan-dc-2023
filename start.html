Starter Kit

Please see the GitHub repository for the competition starter kit, including code for loading the datasets, training baseline detectors and evasive Trojans, and creating a submission.

Accessing the Data

The dataset of networks for each track can be accessed here or through the dataset downloading script in the starter kit.

Submissions

We manage submissions and leaderboards through several linked CodaLab competition pages. All participants are required to register and agree to the rules before submitting.

  • Detection Track & Final Round: https://codalab.lisn.upsaclay.fr/competitions/5951
  • Analysis Track (Target Label Prediction): https://codalab.lisn.upsaclay.fr/competitions/5952
  • Analysis Track (Trigger Synthesis): https://codalab.lisn.upsaclay.fr/competitions/5953
  • Evasive Trojans Track: https://codalab.lisn.upsaclay.fr/competitions/5954
